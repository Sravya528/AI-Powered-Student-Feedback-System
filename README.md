# AI-Powered Student Feedback System

## Overview
An AI-powered web application developed using Java and Spring Boot that allows students to submit feedback and enables administrators to analyze feedback through a dashboard.

## Features
- Student Login
- Admin Login
- Submit Student Feedback
- Admin Dashboard
- H2 Database Integration
- AI-based Sentiment Analysis

## Technologies Used
- Java 17
- Spring Boot
- Spring Data JPA
- Maven
- H2 Database
- HTML
- CSS
- JavaScript
- Git & GitHub

## How to Run

Clone the repository

```bash
git clone https://github.com/Sravya528/AI-Powered-Student-Feedback-System.git
```

Run the project

```bash
mvn spring-boot:run
```

Open in your browser

```
http://localhost:9090
```
